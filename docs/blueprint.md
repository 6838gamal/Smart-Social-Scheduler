# **App Name**: Smart Social Scheduler

## Core Features:

- Dashboard Overview: Provides a visual summary of scheduled, published, and failed posts.
- Multi-Platform Scheduling: Schedule posts to multiple platforms including Email, WhatsApp, Telegram, and Notion.
- Template Management: Allows users to create, save, and apply pre-designed templates for quicker post creation.
- Analytics Reporting: Generates reports on post performance, identifying optimal posting times using generative AI to look for trends.
- Account Management: Enables linking and managing multiple social media accounts from a unified interface.
- Language Support: Supports both Arabic (RTL) and English (LTR) with easy switching via a topbar button.
- AI-Powered Content Tool Suggestions: Suggests hashtags to include, or times to consider based on similar posts with high user engagement using an AI tool.

## Style Guidelines:

- Primary color: Light blue (#64B5F6) to evoke a sense of calm and reliability.
- Background color: Very light blue (#F0F8FF), almost white, providing a clean and spacious feel.
- Accent color: Soft purple (#BA68C8) for interactive elements and highlights, adding a touch of sophistication.
- Body font: 'PT Sans', a modern and versatile sans-serif font suitable for both Arabic and English.
- Headline font: 'Space Grotesk', a techy sans-serif font for headings. Note: currently only Google Fonts are supported.
- Use Material Design icons for a consistent and recognizable visual language.
- Responsive design adapting to desktop, tablet, and mobile with sidebar navigation on larger screens and bottom tabs on mobile.
- Subtle transitions and animations to enhance user experience, especially when navigating between tabs and sections.
